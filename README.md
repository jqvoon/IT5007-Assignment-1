# IT5007-Assignment-1

TicketMaster - Design a reservation system for limited-seat concert event happening in Singapore.

Requirements:
1) A Single Page Application (SPA) with the long-term goal
of supporting better responsiveness
2) System should allow users to the following functionality:
  i) reserve tickets,
  ii) view attendee lists,
  iii) cancel reservations, and
  iv) visually see remaining availability
